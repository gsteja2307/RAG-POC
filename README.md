# RAG POC

## Prerequisites
- Python 3.8+ installed on your system.

## Setup Instructions (Windows)

1. **Activate the Virtual Environment:**
   From the command prompt, located in the project's root directory (`c:\CODE\RAG POC`), run:
   ```cmd
   venv\Scripts\activate
   ```
   > You should see `(venv)` appear before your command prompt line, indicating the virtual environment is active.

2. **Install Dependencies:**
   With the virtual environment active, run the following command to install the project dependencies:
   ```cmd
   pip install -r requirements.txt
   ```

3. **Set Up the Environment Variables:**
   Create or modify the `.env` file at the root of the project with the necessary environment configurations (such as your Google API Key).
